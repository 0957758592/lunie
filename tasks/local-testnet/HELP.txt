# Certificate

Important use localhost as Common Name on command 2:
```
Common Name (e.g. server FQDN or YOUR name) []:localhost
```

Run:
```
openssl genrsa -out certs/key.pem 4096
openssl req -new -key certs/key.pem -out certs/csr.pem
openssl x509 -in certs/csr.pem -out certs/cert.pem -req -signkey certs/key.pem -days 365
```
